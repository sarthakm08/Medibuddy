# **App Name**: Medibuddy

## Core Features:

- Intelligent Report Uploader: Securely upload PDF or image medical reports for automated data extraction.
- AI Diagnostic Analysis: An AI tool that parses complex medical jargon from reports to identify medications, dosages, and treatment timelines.
- Patient Profile Management: Interface for users to enter personalized data such as specific allergies, height, weight, and chronic parameters.
- Cross-Referenced Medication Summary: Utilizes a reasoning tool to compare extracted medicine data against the patient's recorded allergies and parameters for potential warnings.
- Unified Health Dashboard: A minimalist overview of current medications, scheduling instructions, and clinical insights tailored to the individual.
- PDF Summary Export: Generates a comprehensive, printer-friendly summary of the analysis and patient profile.

## Style Guidelines:

- Primary color: Clinical Blue (#2855BE) used for core action items, reflecting professional stability and healthcare standards.
- Background color: Pure Off-White (#F7F8FA), a subtle desaturated tint of the primary to ensure a clean, minimalist feel.
- Accent color: Vibrant Azure (#166F9C) to highlight important alerts or distinct data parameters without being overwhelming.
- Font pairing: 'Space Grotesk' (sans-serif) for headlines to provide a modern, technical precision, paired with 'Inter' (sans-serif) for body text to ensure maximum readability in dense medical summaries.
- Line-based, minimal icons with consistent weight to match the objective and machine-efficient styling of the brand.
- Spacious card-based layout with significant negative space to avoid cognitive overload when presenting health data.
- Delicate opacity transitions when shifting between upload, input, and final summary states to maintain a calm user experience.
